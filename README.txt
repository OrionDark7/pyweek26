Package requirements:
- Pygame 1.9.x, preferably 1.9.4
- Python 3.x or Python 2.7.x, the Source version works on either one.

To Run Game:

Run the File "main.py" either in python or pythonw

To run it from the command line, type:
python /location/of/install/main.py
